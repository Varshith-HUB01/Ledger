Ledger Link Engine

A distributed, production-ready URL shortener service built with a focus on observability, security, and tactile UI/UX.

 Architecture Overview
* API: Go (Golang) with `chi` router
* Authentication: JWT (JSON Web Tokens) & Bcrypt
* Database: PostgreSQL (Persistent storage)
* Cache: Redis (Fast lookup)
* Message Queue: RabbitMQ (Asynchronous event logging)
* Observability: Prometheus & Grafana
* Frontend: React (Vite) with custom Neumorphic CSS

 Getting Started

Prerequisites
- Docker & Docker Compose installed on your machine.
